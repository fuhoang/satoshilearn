---
title: The Blockchain Explained
---

The blockchain is Bitcoin's public history of transactions, grouped into blocks and linked together in order. You can think of it as a ledger made of pages, where each new page includes a reference to the page before it. That linking creates a chain of blocks.

Each block contains a batch of valid transactions, a timestamp, and a reference to the previous block. Because each block points backward, changing older records becomes extremely difficult. If someone tried to alter one block, they would need to rebuild everything after it and convince the rest of the network to accept the change.

The blockchain matters because it gives the network one shared source of truth. Instead of every user trusting one bank's private database, participants can verify the same public history themselves. This is how Bitcoin avoids the problem of digital copying and double spending.

People sometimes imagine the blockchain as something mysterious, but the core idea is simple. It is a tamper-resistant, ordered record that many participants can inspect. Blocks are just containers. The chain is just the sequence that connects them.

In Bitcoin, the blockchain is not valuable because it is a buzzword. It is valuable because it helps the network preserve a single agreed history of ownership without depending on one central authority.
